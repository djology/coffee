Trail: Getting Started（基础入门)

The Java Technology Phenomenon（Java技术现象）
    关于Java语言本身

            Java语言

                    在Java编程语言中，所有源代码首先编写在.java扩展名结尾的纯文本文件中，
                    然后这些源文件被编译器javac编译成.class文件，.class文件中包含字节码 --- 是Java虚拟机的机器语言

                    Java VM可用于不同的操作系统，所以相同的.class文件可在不同操作系统中运行

            Java平台

                    大多数平台是指操作系统和底层硬件的组合，Java平台是一个纯软件平台，运行在其他基于硬件的平台之上

                    Java平台的两大组件：

                            Java虚拟机
                            Java API

                    Java虚拟机是Java平台的基础，并被移植到各种基于硬件的平台上
                    API是大量现成的软件组合，被分成相关类和接口的库，这些库称为包


    Java的技术堆栈

            开发工具
                    提供编译、运行、监控、调试和记录应用程序所需的一切

            应用程序编程接口（API）
                    涵盖从基本对象到网络和安全性，再到XML生成和数据库访问等所有内容

            部署技术
                    JDK软件提供标准机制，以用于将应用程序部署到最终用户

            用户界面工具包
                    JavaFX、Swing和Java2D工具包使创建复杂图形用户界面（GUI）成为可能

            集成库
                    通过Internet间ORB协议技术等集成库支持数据库访问和操纵远程对象
